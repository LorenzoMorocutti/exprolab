Description:
This node implements the the finite state machine, with the smach library, that manage the overall application. The logic is developed in four states and it's very simple: I go to a new room (independently of where I am, in this case); then I look for a hint; if I have enough hint, I go home and try to make a hypotesys; if the hypotesis is consistent and complete, I try to guess the killer, where the murder has happened and with which weapon.
