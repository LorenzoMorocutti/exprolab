Description:
This node implements the the finite state machine, with the smach library, that manage the overall application. The logic is developed in four states and it's very simple: I go to a new room (the robot has to check all the rooms and can't return to the same room twice) setting as a goal pose one of the rooms; then I look for a hint (read all the markers near the robot) moving the arm, in order to catch all the markers around the robot; if I have enough hint and a complete hypotesys, I go home; if the hypotesis is consistent and complete, I try to guess the killer, where the murder has happened and with which weapon.
